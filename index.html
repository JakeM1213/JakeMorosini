The Library of Babel

“The Library of Babel” by Jorge Luis Borges envisions an infinite library with every possible combination of letters, encompassing all texts from profound to nonsensical. The story examines knowledge, meaning, and humanity's search for truth. Inspired by this concept and the misinformation during the COVID pandemic, I recreated the "library" as an infinite Twitter-like website. Borges’ texts represent "truth," while GIFs and other content symbolize chaos. The site includes infinite scrolling, interactive elements like a like and comment button, and hidden easter eggs. Coded using HTML, CSS, and JS. 2024.

The Witching Hour

The Witching Hour. I made a website based on time. I decided to create my website around the witching (devils) hour. From 4:01 AM - 2:59 AM the website is normal. An eery, unsettling video plays, however from the times of 3:00-3:59 AM the website changes - it goes red, with cryptic text, as if it just crashed. Coded using HTML, CSS, and JS. 2024.
